# What's New in RADCounter 5.1.1

## New: a documentation website, linked right from the app
RADCounter now has a full documentation site — every feature explained, browsable by category, workflow stage, or A–Z, with search.

- Open it from **Settings → About → "Documentation & feature guide"**, or grab the address with the **Copy link** button next to it.
- Live at **https://erichter2018.github.io/RADCounter-dist/**.
- It's a companion to the MosaicTools docs site, and the two link to each other.

## New: fresh app icon
RADCounter has a new icon — four ascending bars with a rising "pace" pulse, in a cool blue gradient. You'll see it on the taskbar, in Alt-Tab, and on the window.

There are no functional or data changes in this release — it's the documentation link, the icon, and the site going public.
