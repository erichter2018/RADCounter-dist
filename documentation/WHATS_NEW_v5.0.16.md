# What's New in RADCounter 5.0.16

## Start automatically when Windows starts
A new **Startup** section at the top of **Settings → General** lets you launch RADCounter automatically when you log in to Windows.

- **Off by default** — enable the **"Start automatically when Windows starts"** checkbox to turn it on.
- **No administrator rights required** — uses your per-user startup entry.
- **Survives moves and updates** — the startup entry is refreshed to the app's current location each time it launches, so it keeps working after an in-place update or if you move the app's folder.
