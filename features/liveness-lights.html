Integration Status Lights

The bottom status bar is three at-a-glance lights — Clario CDP, Mosaic CDP, MosaicTools — green when live, red when down.

What it does

The bottom status bar is a set of three liveness lights — Clario CDP · Mosaic CDP · MosaicTools — each green when the connection is live and red when it's down, so you can tell at a glance which data sources RADCounter has.

Using it

Hover any light to see its exact state in a tooltip. Right-click the bar for a quick jump to the CDP settings and a "refresh liveness" action.

Tips & gotchas

  • The lights update on their own every couple of seconds — a light going red is your cue that a source dropped (e.g. Clario was closed or MosaicTools restarted).
  • A red Mosaic CDP light is only meaningful when you're relying on it as a fallback; while MosaicTools is up, the bridge is what's used.
