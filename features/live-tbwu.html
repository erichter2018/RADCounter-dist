Live TBWU Tracking

The time-based work-unit total and rate — RADCounter's headline productivity metric — live on the main bar.

What it does

TBWU (time-based work units) is the weighting RADCounter uses to reflect the effort a study actually takes. It shows your TBWU total and TBWU/hour live, and can display secondary readouts like This Hour, Prev Hour, Est Total, TBWU/Study, and Pace.

How to turn it on / set it up

Pick which metrics appear under Settings → Display. The current dollars-per-TBWU rate that drives income projections is set under Settings → Compensation → Current $/TBWU rate.

Using it

Each study's TBWU comes from the bundled value tables, adjusted by patient class (ER / inpatient / outpatient). Because TBWU is rate-aware, it's the number RADCounter uses for pace and projected income.

Tips & gotchas

  • Est Total projects your end-of-shift TBWU from your current rate — handy mid-shift, noisy in the first hour.
  • RADCounter also reads Clario's own per-study work unit so you can compare the two — see [[clario-wu-compare]].
