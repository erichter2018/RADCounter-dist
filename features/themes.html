Themes

Twenty-plus color presets plus a custom theme, with dark and light options and an always-on-top mode.

What it does

RADCounter ships with a large set of color themes — twenty-plus presets across dark and light — plus a custom option, so the compact bar fits your reading setup.

How to turn it on / set it up

Pick a theme under Settings → Appearance. On the General tab, Always on top keeps the bar visible over your worklist and Start minimized tucks it away on launch.

Using it

Theme changes apply immediately. Because the main window is a slim bar, a theme that reads well at a glance in your reading-room lighting is the one to keep.

Tips & gotchas

  • Show time in display (Display tab) adds the clock to the bar if you want it there.
  • Themes are cosmetic only — they don't affect any counting or data.
