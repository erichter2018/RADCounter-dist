CDP Setup

One-click provisioning that sets up the Clario and Mosaic CDP connections RADCounter uses to read work units and the current study.

What it does

The CDP tab gets RADCounter's direct connections to Clario and Mosaic running. Each has an enable toggle, a one-click Set up button that provisions the connection, and a live status readout.

How to turn it on / set it up

  1. Open Settings → CDP.
  2. For Clario: check Enable Clario CDP (work-unit comparison), click Set up Clario CDP, then launch Clario from the shortcut it creates.
  3. For Mosaic (optional fallback): check Enable Mosaic CDP reader, click Set up Mosaic CDP, and restart Mosaic once.
  4. Watch the status lines — and the bottom-bar lights — go green.

Using it

Setup is a one-time step per machine. After that, RADCounter reconnects on its own each session. Both readers are off by default, so nothing changes until you opt in.

Tips & gotchas

  • The Clario reader powers [[clario-wu-compare]], [[clario-signed-time]], and [[patient-class]]; the Mosaic reader is the MosaicTools-down fallback ([[mosaic-cdp]]).
  • Connection health is always visible in the [[liveness-lights]] at the bottom of the window.
