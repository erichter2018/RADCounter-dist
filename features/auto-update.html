Auto-Update

RADCounter checks for new versions on startup and updates itself in place — no installer.

What it does

RADCounter keeps itself current. On startup it can check for a newer release and update in place, so you're always on the latest version without running an installer. A "What's New" summary shows what changed.

How to turn it on / set it up

Leave Settings → General → Automatically check for updates on startup on (the default). You can also check manually from the About tab.

Using it

When an update is found, RADCounter fetches it and swaps itself over on the next launch. Your settings and local history carry across untouched.

Tips & gotchas

  • It's a single self-contained exe — updating replaces that one file.
  • "What's New" is fetched fresh each release, so you can always see what changed in the version you just got.
