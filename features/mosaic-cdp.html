Mosaic CDP Reader

Reads the current study straight from Mosaic when the MosaicTools bridge is down — a more reliable fallback than the accessibility tree.

What it does

If the MosaicTools bridge isn't available, RADCounter can read the current study directly from Mosaic over the Chrome DevTools Protocol — a more reliable identification of what you're reading than scraping the accessibility tree alone.

How to turn it on / set it up

Enable Settings → CDP → Enable Mosaic CDP reader (used only when MosaicTools isn't running), run Set up Mosaic CDP, and restart Mosaic once. Off by default. Full walkthrough in [[cdp-setup]].

Using it

This is strictly a fallback: while MosaicTools is running its bridge is used exactly as before. The reader only kicks in when the bridge is down, and the bottom-bar Mosaic CDP light shows whether it's connected.

Tips & gotchas

  • It handles both the newer and classic Mosaic viewers.
  • Because it's a fallback, most users leave it enabled and never notice it — it quietly covers a MosaicTools restart.
