# RADCounter-dist

Public distribution repository for [RADCounter](https://github.com/erichter2018/RVUCounter) (RVU/TBWU tracker for radiology).

This repo holds release artifacts (`RVUCounter.exe`, `RVUCounter.zip`) consumed by the in-app auto-updater. Source code lives in the private `RVUCounter` repo.

## Install

Download the latest `RVUCounter.zip` from [Releases](https://github.com/erichter2018/RADCounter-dist/releases/latest), extract, and run `RVUCounter.exe`.
